Basics of HTML

What is HTML?

HTML stands for Hypertext Markup Language. This markup language is used to create documents on the world wide web by using html tags and attributes. A web document is viewed in a web browser (chrome, mozilla, internet explorer, etc).


HTML editors

  • Notepad++
  • Sublime Text3
  • Brackets

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
<title>TeamBee Tutorials</title>
</head>
<body>
<h1>TeamBee</h1>
<p>HTML5 Tutorial</p>
</body>
</html>

  • The <!DOCTYPE html> declaration defines this document to be HTML5
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Displaying an html Document

HTML documents can be viewed using a web browser. Web browsers can detect errors that are made in an html document. After creating or as you create your web page, you can make necessary changes if your web page does not look the way you expect.

Creating Headings

Headings are very important in a web page because it tells what a section is about. There are six headings in HTML, <h1> to <h6>. <h1> is the largest and <h6> is the smallest. Each heading can be created using a header tag. Use the following syntax to create a header tag: <h1>Heading 1</h1>

Creating Paragraphs

The P element is one of the most common and most used HTML element. It creates new lines for the block of text. In HTML, creating paragraph are very simple. Paragraphs can be created using a p tag. Use the following syntax to create a paragraph: <p>This is a paragraph.</p>

Creating Links

The A element is used to create links which are used for navigation. You can click on a link and jump to another document or page. In HTML, links are defined using an <a> tag. Use the following syntax to create a link: <a href="url">This is a Link.</a>
The href attribute specifies the destination of the link.

Creating Lists

Lists are used to group information so that it is easy to read and understand. There are three types of list in HTML.

  • unordered list - used to group a set of related items in no particular order. Use the following syntax:
    <ul><li>n</li></ul>
  • ordered list - used to group a set of related items in a specific order.
    <ol><li>n</li></ol>
  • description list - used to display name/value pairs such as terms and definitions.
    <dl><dt>Name</dt><dd>Value</dd></dl>

HTML Tables

Tables are created to arrange webpage data like texts, images, links etc. into specific rows and columns. An HTML table is defined using the <table> tag.

<tr> - table row.

<th> - table header.

<td> - table data or cell.

	<table border="solid">
		<tr>
			<th>HTML</th>
			<th>CSS</th>
			<th>JS</th>
		</tr>
		<tr>
			<td>Hypertext Markup Language</td>
			<td>Cascading Stylesheets</td>
			<td>Javascript</td>
		</tr>
	</table>
