CSS Selectors

  • Selector - is an HTML tag wherein it is used in CSS rule to determine which style will be applied.
    • Type Selectors or Element Selector - The most easy to understand selector. . It will select all matched elements in the document.

      CSS Code

      Result


    • Grouped Selector - Elements with the same style are combined into one declaration block wherein selectors are separated with a comma.
      Example Grouped Selector


    • ID selectors - An ID Selector styles an element based on the value of its designated ID attribute. It is preceded by a number sign(#) and it can only identify one element.

    • Class selectors -A Class Selector is preceded by a period mark(.) and it can be used to identify two or more elements.

    • Universal selectors - A universal selector has a asterisk(*) that will match any element in the document.

    • Attribute selectors - used to style elements based on values or attributes.
      • [attribute=value] - Selects all elements with target="_blank"
      • [attribute~=value] - Selects all elements with a title attribute containing the value you put inside the attribute.
      • [attribute !=value] - Select elements that either don't have the specified attribute.
      Attribute Selectors is only supported in the latest versions of browsers.


    • Pseudo-classes - a keyword added to the end of a selector, which is used to specify the element you want to style.
      • :active - Matches link that are in the state of being actived.
      • :focus - Keyboard input focus.
      • :hover - Matches link when hovered by the cursor.
      • :after - inserts content after an element.
      • :before - inserts content before an element.
      • :first-child - styles an element that is the first child of another element.
      • :first-letter - styles the first character.
        • clear
        • background properties
        • border properties
        • float
        • font properties
        • padding properties
        • margin properties
        • text-decoration
      • :first-line - styles the first line of the text. The following properties are only applied to the :first-line pseudo-element.
        • clear
        • background properties
        • color properties
        • font properties
        • word-spacing
        • letter-spacing
        • text-decoration
      • :lang - styles elements with a specific language attribute.
      • :link - Matches link that are not yet visited.
      • :visited - Matches link that are already visited.
      • :


    • Combinators - relationship between selectors.
      • Descendant Selectors - It is made up of two or more selectors with whitespace between two selectors.
        • whitespace
        • space
        • tab
        • line feed
        • carriage
        • return
        • form feed
        The white space is a combinator that specifies the ancestor of some element.


      • Child Selectors (>) - Like descendant selectors it is made up of two or more selectors but separated by ">".

      • Sibling Selectors
        • Adjacent Sibling Selector - separates two selectors with plus sign(+).
        • General Sibling Selector - separates two selectors tilde symbol(~).
