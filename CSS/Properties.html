CSS Properties

  • Properties
    • Shorthand Properties - allows authors set several property value in a single line
      • border-width
      • background
      • font
      • margin
      • padding

    • vendor specific extensions (a.k.a. vendor prefixes)- used by browser vendors as a prefix for the names of experimental or non-standard CSS properties.
      • -webkit-
      • -moz-
      • -o-
      • -ms-

    • custom properties a.k.a, CSS variables - Property names prefixed with "- -", that contains a specific value that can be used again in every part of the document.
